Ver1.0
################使い方###########################################


1. 画像置き場フォルダにポケカ公式検索サイトから持ってきた「*.jpg」を置く
　→ 複数でもOK（１度の実行でフォルダ内のjpgを全て処理）
　　　注意: jpgじゃないとダメです

2. 「yosuminuki」のエイリアスファイル（真っ黒のやつ）をダブルクリックで実行
　　→ 最初はダブルクリックでは実行できないかも→その場合は右クリックから開く
　　→ エイリアスが動かない場合は/dist/yosuminuki/yosuminukiの実行ファイルを実行
　　→ 右クリックからエイリアスの作成で好きな場所に実行ファイルおいてもいい

3. 白枠抜きフォルダに４隅を透過させた画像
　　白枠ありフォルダに白枠を加えた画像　　　　がそれぞれ保存される
   ※ 処理された元画像は削除されます。


###############注意点#############################################

・実行ファイルがあるフォルダ「四隅抜き」は絶対デスクトップに直に置いてね
　→ 絶対パスをデスクトップに指定しているので、それ以外に置くと起動しない

・四隅抜きフォルダ内のファイルやフォルダの位置は動かさないでね。名前も変えないで（四隅抜きって名前も）
　→ 同上の理由
